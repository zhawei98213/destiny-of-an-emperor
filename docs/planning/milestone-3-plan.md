# Milestone 3 Plan / 第三阶段计划

中文：第三阶段执行“小沛城镇内部 + 证据门槛制原版推断桥”。先做 metadata-only ROM/town probe 和证据分类，再实现可玩的 prototype/证据标记城镇切片。  
English: Milestone 3 implements “Xiaopei town interior + evidence-gated original-inference bridge.” It first adds metadata-only ROM/town probing and evidence classification, then implements a playable town slice labeled as prototype or evidence-backed.

## Chosen route / 选择路线

中文：采用证据门槛制。没有明确证据时，不宣称内容来自原版；仍交付可玩的 prototype 城镇。  
English: Use an evidence gate. Without explicit evidence, content is not claimed to be original-derived; a playable prototype town is still delivered.
