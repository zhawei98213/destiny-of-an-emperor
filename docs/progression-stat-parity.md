# Progression Stat Parity
# 成长曲线一致性

This audit tracks whether imported chapters keep party growth, enemy strength, equipment access, and reward curves within a reviewable range.
这份审计用于跟踪已导入章节是否让队伍成长、敌人强度、装备获取和奖励曲线保持在可审查范围内。

Outputs:
输出：

- party growth report
- level / stat curve summary
- equipment progression summary
- obvious outlier detection

Command:
命令：

- `npm run progression-stat-parity`
