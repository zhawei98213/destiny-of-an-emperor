# Visual Backfill Reports
# 视觉回填报告

This directory stores generated visual backfill comparison reports and checklists.
此目录用于存放生成式视觉回填对比报告和清单。

Expected files:
预期文件：

- `latest/<plan-id>.report.json`
- `latest/<plan-id>.summary.md`
- `latest/<plan-id>.checklist.md`

These artifacts are generated by:
这些产物由以下命令生成：

- `npm run visual-backfill-report -- --id <plan-id>`
- `npm run visual-backfill-checklist -- --id <plan-id>`

Do not edit files under `latest/` by hand.
不要手工编辑 `latest/` 下的文件。
