# Battle Calibration Reports
# 战斗校准报告

`npm run battle-enemy-group-import`, `npm run battle-scenario-import`,
`npm run battle-reward-drop-check`, and `npm run battle-parity-checklist`
write their latest artifacts here.

`npm run battle-enemy-group-import`、`npm run battle-scenario-import`、
`npm run battle-reward-drop-check` 和 `npm run battle-parity-checklist`
会把最新产物写到这里。

The files are review-oriented and should be used alongside `npm run battle-parity`.
这些文件面向审查，应与 `npm run battle-parity` 一起使用。
