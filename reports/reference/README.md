# Reference Reports
# 参考资料报告

This directory stores validator output for the reference pipeline.
这里存放 reference pipeline 的校验输出。

`latest/report.json` contains the full machine-readable result.
`latest/report.json` 包含完整的机器可读结果。

`latest/summary.md` contains the short review summary.
`latest/summary.md` 包含简短的审查摘要。
