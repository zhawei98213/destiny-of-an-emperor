`reports/chapters/latest/` 是章节 completeness、状态和锁定清单报告的生成输出目录。

Current generated outputs include:
当前生成产物包括：

- `completeness-report.json` 和 `completeness-report.md`
- `status-report.json` and `status-report.md`
- `status-report.json` 和 `status-report.md`
- `lock-checklists/<chapter-id>.md`
