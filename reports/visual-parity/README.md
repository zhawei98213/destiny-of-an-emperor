# Visual Parity Reports
# 视觉一致性报告

This directory stores visual backfill scoring artifacts generated by `npm run visual-parity-score -- --id <plan-id>`.
这个目录用于存放通过 `npm run visual-parity-score -- --id <plan-id>` 生成的视觉回填评分产物。

The generated `latest/` directory is ignored by Git.
生成的 `latest/` 目录已被 Git 忽略。

Use these reports together with:
请将这些报告与以下产物配合使用：

- `reports/visual-backfill/latest/`
- `reports/asset-parity/latest/`
- `reports/ui-parity/latest/`
- `reports/regression/latest/`

The intent is to compare registry-managed placeholder baselines against reconstructed chapter overrides without changing gameplay content data.
其目标是在不修改 gameplay 内容数据的前提下，对比由 registry 管理的 placeholder 基线与重建后的章节级 override。
