# UI Asset Reconstruction Reports
# UI 资产重建报告

`reports/ui-asset-reconstruction/latest/` stores the latest structured report for the UI asset reconstruction workflow.
`reports/ui-asset-reconstruction/latest/` 保存 UI 资产重建流程的最新结构化报告。

Generated files:
生成文件包括：

- `report.json`: machine-readable reconstruction summary
- `report.json`：供工具读取的重建摘要
- `summary.md`: short human-readable overview
- `summary.md`：给人工快速阅读的简要摘要

Do not hand-edit files inside `latest/`.
不要手工修改 `latest/` 目录中的文件。
