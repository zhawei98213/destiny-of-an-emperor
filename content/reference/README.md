# Reference Content
# 参考资料目录

This directory stores traceable visual or editorial reference inputs used for parity and
future asset reconstruction work.
这里存放可追溯的视觉或编辑参考输入，用于一致性核对和后续资产重建。

Reference inputs are **not** runtime assets.
参考输入**不是**运行时资源。

Do not load files from `content/reference/` in `game/`.
不要让 `game/` 直接加载 `content/reference/` 下的文件。

All references should be indexed through `content/reference/manifest.json`.
所有参考资料都应通过 `content/reference/manifest.json` 建立索引。
