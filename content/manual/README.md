# Manual Content
# 手工内容

`content/manual/` contains reviewed or hand-authored runtime content.
`content/manual/` 用于存放经过人工审阅或直接手工编写的运行时内容。

Use this directory for overrides, corrections, and authored data that should remain under direct editorial control.
这个目录适合放需要人工把关的修正内容、覆盖内容以及希望直接编辑维护的数据。

Chapter-level metadata lives under `content/manual/chapters/`.
章节级元数据位于 `content/manual/chapters/`。
