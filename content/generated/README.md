# 工具生成内容

`content/generated/` is tool output.
`content/generated/` 是工具生成输出目录。

Do not hand-edit files here.
不要手工修改这里的文件。

Re-run `npm run import:all` instead.
需要更新时请重新执行 `npm run import:all`。

Only manifest-listed `*.content.json` files are runtime-consumable by the game. Staging reports in this directory are importer artifacts for inspection only.
只有 manifest 中列出的 `*.content.json` 文件会被游戏运行时消费。这个目录里的 staging report 只是导入器产物，用于检查，不会被游戏直接读取。
