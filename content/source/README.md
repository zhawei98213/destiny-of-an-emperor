# Source Content
# 原始内容

`content/source/` stores raw importer inputs.
`content/source/` 用于存放导入工具的原始输入。

Do not point the game runtime at this directory.
不要让游戏运行时直接读取这个目录。

These files may be extracted, exported, or hand-curated source tables that still need tooling before they become runtime content.
这些文件可以是提取数据、导出表格或人工整理过的原始资料，但它们在成为运行时内容之前仍然需要经过工具处理。
